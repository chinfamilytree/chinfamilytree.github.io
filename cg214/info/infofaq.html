Q1. What simple text processor should I use to create HTML pages?
A1. If you are running Windows, use Notepad. If you are on a Mac, use TextEdit.

In Mac OS X start TextEdit and change the preferences in the New Document tab and the Open and Save tab as outlined below:

  1. New Document
    1. Choose the "Plain Text" option button.
    2. Check the "Wrap to Page" checkbox.
    3. Change the "Plain text font" to a courier font such as Courier New Regular 14 pts.
    4. In the Options area, check the "Check spelling as you type" checkbox. All other check boxes should be off.
  2. Open and Save
    1. Check the "Ignore rich text commands in HTML files". Don't check the "Ignore rich text commands in RTF files".
    2. Check the "Delete the Automatic backup file" checkbox. Don't check "Add .txt extension to plain text files". Click off the Add .txt Extension To Plain Text Files.
    3. Change Document Type to "XHTML 1.0 Strict" for future compatibility.
    4. Don't check the "Preserve white space" checkbox.
    5. Close the Preferences dialog box.
  3. Close the current TextEdit Untitled document since this has the old preferences.
  4. Start a new document (File/New). This new document has the preferences you created above. Note: Don't hide extensions when you save your documents.

Q2. How do I prevent a background image from tiling?
A2. Insert the following code snippet in the <head> section. This code will also center the image specified in the url.

	<style type="text/css">		<!-- internal style sheet -->
		body 	{background-image: url(bkgd.jpg);
      			 background-repeat: no-repeat;
			 background-position: center center;
			}
	</style> 

background-position may have the following values:
	top left
	top center
	top right
	center left
	center center
	center right
	bottom left
	bottom center
	bottom right
	x% y%
	xpos ypos

Q4. How do I change the font color and size in a link when the mouse hovers over it?
A4. In a Cascading Style Sheet (CSS) set the color of the selector anchor a to one color and set the hover anchor pseudo-class to another color. The following CSS sample sets an unvisited link color to #0099FF (light blue) and when you mouse hovers over the link it will turn #FF0000 (red). It also increases the size of the font by 20% when you hover over the link.

CSS Sample:

	body {	background-color: #CCFF99; color:#009900; text-align: center;}

	a {	font-family: Verdana; font-size: medium;
		color: #0099FF;}

	a:link {text-decoration: none;}
	a:visited {text-decoration: none; color: #0099FF;}
	a:hover {text-decoration: none; color: #FF0000;font-size: 120%;}
	a:active {text-decoration: none; color: #00FFFF;}

	p,td,th {color: #009900; font-family: Verdana; font-size: medium;}

Q5. How can I indent text in HTML?
A5. HTML does not have an INDENT tag. However, you can trick it by using the <ul> tag without <li>.

    Example:	
		<ul>
		This text is indented.
		</ul>

Q7. How can I find the size of my web site?
A7.

  1. Right-click on the "Start" button on the left side of the Taskbar.
  2. Click on "Explore".
  3. Expand the drive that contains you web site (by clicking on the "+" sign on the left side of the drive name).
  4. Continue expanding until the folder that contains your web site appears.
  5. Right-click on this folder.
  6. Click on "Properties". The "General" tab shows the size of your site.

Q9. How do I prevent my images from appearing in Google search results?
A9. Create a robots.txt file in the root directory of your site containing the following line:

user-agent:Googlebot-Image disallow: /

Q11. How can I make my background image stay in place while the rest of my text and images scroll over it?
A11. To make the background image fixed in one place add the following attribute to the <body> tag:

bgproperties="fixed"

For example:

		<body background="bgimage1.jpg" bgproperties="fixed">

Another way of fixing the background image is using CSS. Insert the following line into the internal style sheet for A2 above as the fifth line:

		background-attachment: fixed;

Q12. How can I change the leading, kerning/tracking, and word spacing of paragraphs?
A12. Leading, kerning/tracking, and word spacing can all be done with styles. For leading use the line-height property. For kerning/tracking use the letter-spacing property, which may be a negative value. For word spacing use the word-spacing property.

Sample Leading, Letter Spacing, and Word Spacing
Code Explanation Effect
style="font-size:14px;" Normal:

style="font-size:14px; line-height:130%" Leading:

style="font-size:14px; letter-spacing:1px" Kerning/Tracking:
The paragraph to the right has a font size of 14px with letter spacing set to 1px.		 Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris in eros in leo facilisis blandit. Nam posuere laoreet imperdiet. Nam at libero nulla.

Q13. How can I put an icon on the left side of a tab or address bar?
A13.

  1. First you need to create your icon by using an icon maker software such as Online Icon Maker. The address for Online Icon Maker is
    http://www.rw-designer.com/online_icon_maker.php
  2. Put the icon you created into the same folder as index.html. It must be named favicon.ico
  3. Put this line in the head section of index.html:
    <link rel="shortcur icon" href="favicon.ico" type="image/x-icon" />

Q15. How do I create a new tab instead of a new window?
A15. As of 8/11/2014 it appears that opening in a new tab is a functionality of the browser. In both cases the new page opened up in a new tab.

	     <a href="http://www.google.com " target="_blank"> Google in a New Tab

Q16. How do I set Windows to show file extensions?
A16. Click on the Start button > Control Panel > Appearance and Personalization > Folder Options >View tab. In the View tab dialog box uncheck Hide Extension for Known File Types. Click OK.

Q17. How do I zip a file, files, or a folder?
A17. Right click on a file, a bunch of selected files, or a folder, then click on Sent to...Compressed (zipped) folder. A new file is created with the zip extension. The zipped file is usually smaller in file size and not usually be corrupted if emailed.

Q18. Where is my "T" drive?
A18. The "T" (Transport) drive is used to share files/folders between the student and the instructor. You can access the "T" drive by double clicking on the Connect to Class Share icon on the desktop and then type in the instructor's first name, followed by an "_" and then the last name. The "T" drive now shows up on the desktop. Double clicking on the "T" drive will show two folders: Dropbox and ffs. The Dropbox is for students to drop files/folders for the instructor. "ffs" stands for "files for students". This folder is used by the instructor to store files/folders to share with students. The "T" drive is accessible from any computer on campus. It cannot be accessed outside of campus. If you wish to share files/folders with the instructor (who is not on campus) it is best to zip the files/folders (See Q17) and email the zipped file to the instructor.
