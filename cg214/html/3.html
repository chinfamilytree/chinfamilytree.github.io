The <img> Tag

<img src="url" alt="text" title="text" width="width" height="height">

Only the src and alt attributes are required.

The img tag is used to insert an image into your Web document. It has no corresponding closing tag. The src attribute is the most important attribute. It specifies the location of the image. The most widely used image formats on the web are gif, jpg, and png. Attributes may be entered in any order. They do not have to be in the order listed above.

Both the alt attribute and the title attribute may be specified. The text in the alt attribute will be displayed if the browser cannot find the image. The text in the title attribute will be displayed when you hover the mouse over the image area. It does not matter whether the browser finds the image or not. The text displayed with alt or title is enclosed in a rectangle, and is called a "ScreenTip". "title" is one of several global attributes that <img> supports. A few other global attributes that is supported by <img> are "class" and "id".

The width and height attributes give a browser the size of an image so that it may reserve space for it and continue rendering the document while waiting for the image data. The specified width and height attributes override the natural size of the image. If the specified width and height were equal to the natural size of the image, the image would be displayed faster. If you do not specify the width and height attributes, the natural width and natural height will be used but it will be slightly slower.

The above image was inserted using the following code:

<img src="boreas-t.jpg" alt="Boreas" title="Painting of Boreas by J.W. Waterhouse">

Since the width and height attributes were not included, the image was rendered using the natural width and height.

Border

A border may surround an image. Use an internal style sheet for the border property. The border property is a shorthand for the three properties: border-style, border-color and border-width.

  • border-style (required)
  • border-color
  • border-width

The border-style values may be none, dotted, dashed, solid, double, groove, ridge, inset, and outset (as shown in the first 9 images below. The border-width of all the images are 1em. except the first one which has no border. The border-color of the dotted and dashed images are coral. The next 6 are borwn and the last 4 are red with a little of blue (#e3001b). The style of the first red one is dotted, solid, double, and dashed. The style of the second red one is dotted, solid, and double. The style of the third red one is dotted and solid. The style of the last red one is dotted. When one value is used in the style it is applied to all 4 sides. When two values are used, the first value is applied to the top and bottom borders and the second value is applied to the left and right borders. When 3 values are used the first value is applied to the top border, the second value is applied to the left and right borders and the third value is applied to the bottom border. When four values are used the first is applied to the top border, the second value is applied to the right border, the third value is applied to the bottom border, and the fourth value is applied to the left border.

Border color may be specified as a color name or hex triplet. For example. the following two style sheets will use "firebrick" as the border color. #B22222 is the hex triplet for firebrick.

<style type="text/css">
	img{border-style:solid; border-color:firebrick; border-width:0.25em;}
</style>

or

<style type="text/css">
	img{border-style:solid; border-color:#B22222; border-width:0.25em;}
</style>

Border width may be specified in any legal unit in HTML such as px, pt, em. For example: 

<style type="text/css">
	img{border-style:solid; border-color:#e3001b; border-width:0.75em;}
</style>

The <hr> Tag

In HTML5 the <hr> tag shows a change in content, probably between sections of a document. There is no corresponding closing tag. Without CSS <hr> by itself renders as a dull thin line. The first <hr> are examples of this. All attributes must be in CSS. The orange <hr> above was created with the following CSS:

hr.hr01 {
    display: block;
    margin-top: 2em;
    margin-bottom: 2em;
    border-style: solid;
    border-color: orange;
    border-width: 4px;
}

The <hr> below was created with the following CSS:

hr.hr02 {
	display: block;
	margin-top: 0.5em;
	margin-bottom: 3em;
	margin-left: 3em;
	margin-right: 3em;
	border: inset bisque 12px;
}

Background image for the <body>

The background image for the body tag is used to provide a background image for a web document. Since all attributes of the <body> were deprecated in HTML 4.01 and not supported in HTML5, you should use CSS for the background image. An example is:

body{background-image:url(image.ext);}

where image is the file name of a picture and ext is the file extension. You may use both the background-image and background-color properties in the same document and if the background-image is not repeated (tiled) then both will appear in the same page. If the background-image is repeated, the background-image will override background-color property. The following example uses ophelia.jpg as the background image and as an image in the document. You can distinguish the two easily since the background image cannot have a border and cannot be resized.

<!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>background image position</title>
<style type="text/css">
  body,td,th {
		font-family: Verdana, Geneva, sans-serif;
		font-size: 14px;
		color: #600;}
	body {
		background-color: #FFC;
background-image: url(ophelia.jpg);
background-repeat: no-repeat;
background-position: top right;}
img{border: ridge brown 1em;}
</style>
</head>
<body>
<h2>A background-color(#FFC) and background-image<br>
(ophelia.jpg) are both used in this file.</h2>
<h2>  background-position is set to top right.</h2>
<p>background-position may have the following values:<br>
top left<br>
top center<br>
top right<br>
center left<br>
center center<br>
center right<br>
bottom left<br>
bottom center<br>
bottom right<br>
x% y%<br>
xpos ypos</p>
<p><img src="ophelia.jpg" width="222" height="333" title=
"This is not the background. It has a border 
and has been resized. The bigger image is the 
background image."></p>
</body>
