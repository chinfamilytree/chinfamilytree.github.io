HTML/CSS Lesson 4 - Links and Link Colors

Links | Link Colors | named anchors |

Links

What separates HTML publishing from traditional publishing is the feature called hyperlink, or web link, or simply link. A link uses the hyperlink tag <a>. The corresponding closing hyperlink tag is </a>. There are three parts to a link: the destination, a label, and a target.

The href (hyperlink reference) attribute of the hyperlink tag specifies the destination, which may be any web resource such as an image, a video clip, a sound bite, or another HTML document. These resources may be on another website, another HTML document within the same website or somewhere within the same document.

Between the opening and closing hyperlink tag is the label, which may be text or an image. The label is what you click on to activate the link or retrieve another web resource.

The target specifies what window or frame is used to display the destination. The default target is itself, i.e., the same window or frame.

Example 1 - Link to another website using text as a label and another window for the target: 

<a href="http://www.google.com" target="_blank">Google</a>

The URL value for this href is an absolute URL. Add the <!doctype html>, <head>, and <body> sections and save as links.html in lesson4 folder.

Example 2 - Link to another website using an image as a label and the default target, which is _self:

<a href="http://www.fitnyc.edu/">
<img src="fitlogo.jpg" width="96" height="96" alt="FIT" title="FIT" />
</a>

The URL value for this href is an absolute URL. Add this line to the links.html document. Separate the two links with a line break <br> and save it again.

Example 3 - Link to another document in the same website:

<a href="links.html">My Sample Links</a>

The URL value for this href is a relative URL. Save this as index.html in lesson4 folder. Test it out to see if all 3 examples work. Notice that the default link color is blue and underlined, the visited link color is purple and underlined.

Link Colors

Notice that the default link color is blue and underlined, the default visited link color is purple and underlined, and the default active link color is red and underlined. Default link colors can be changed by specifying the state (pseudo-classes) of the hyperlink selector. The specification format is:

a:state { property: value; }

where state is the pseudo-class.

CSS Code:

a:link {color:blue;}
a:visited {color:magenta;}
a:hover{color:red;}
a:active{color:lime;}

The order of the specification must be: link first, then visited, then hover, and then active.

Try this HTML code:

<a href="http://www.google.com" target="_blank">Google</a>

The default decoration is underline. To change the default, specify in each of the 4 states which underline style you want.

<style type="text/css">
a:link {color:blue; text-decoration: none;}
a:visited {color:magenta; text-decoration: none;}
a:hover{color:red; text-decoration: underline;}
a:active{color:lime; text-decoration: none;}
</style>

Named Anchors or Anchor URL

The next excerpt is an example of a link within the same document using a named anchor (in other words, the URL value for this href is an anchor URL). An example of an anchor URL is href="#top", where top is the value of an idattribute for any element where an id attribute is valid.

Create a new folder in classwork/lesson4 and name it bible. Save the following excerpt as bible.html in the bible folder. Add a link at the end to go back to the top. Change the default link colors to some other colors. Also change the text color to navy or teal.

<!doctype html>
<html><head><title>Bible</title>
<meta charset="utf-8">  
<style type="text/css">
body {background-color:gray; color:beige;}
a:link {color:blue; text-decoration: none;}
a:visited {color:magenta; text-decoration: none;}
a:hover{color:red; text-decoration: underline;}
a:active{color:lime; text-decoration: none;}
</style>
</head>
