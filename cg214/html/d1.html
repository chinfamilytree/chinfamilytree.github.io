To Define a Site

  1. Launch Dreamweaver.
  2. Choose Site > New Site. The Site Setup For Unnamed Site 1 dialog box appears.
  3. In the Site Name text box, enter the name of your new site.
  4. For the Local Site Folder text box, click the folder icon to browse for the folder where your files are located. Double click the folder that contains your files. Click Select.
  5. Click Save. Your files now appear in the Files panel on the right side of the screen.

To Add Folders and Files in the Files Panel

Files are saved automatically when you create them using the Files panel. Files are not saved automatically if you create them using the File > New or the Start window menu.

  1. Activate the Files panel by clicking on its name in the panel area.
  2. Right-click a folder in which you want to add a file or folder.
  3. Select New Folder or New File. Type in the new folder or file name and press Enter.
  4. If you want to delete a file or folder, right-click the name, select Edit > Delete, and click on Yes to confirm the deletion.
  5. If you want to rename a file or folder, right-click the name, select Edit > Rename, type the new name and press Enter.

  1. Use the above procedure to create a new file.
  2. Type index.html as its name. This file is your default home page.

To Set Page Properties

Page Properties adds embedded styles and only affects the current page. If you have more than a few pages, it is better to use external style sheets (see Dreamweaver Lesson 2).

  1. Double click on index.html if it is not already open.
  2. In the Properties Inspector, change the Title from Untitled Document to Dreamweaver Lesson 1.
  3. Select Modify > Page Properties.
  4. In the Appearance (CSS) category, select a Page Font, Size try to use em), Text Color, Background Color, and Background image, if any. Enter 0 or some small number for the four Margin settings.
  5. In the Links (CSS) category, select the link colors.
  6. In the Headings (CSS) category, set the font size (such as 2.25em, 1.5em, 1.125em, 1em) and color for each heading.
  7. In the Title/Encoding category, type the Title for your document if you have not done this in Step 2 above, choose the Document Type (DTD) as HTML5. Enter the Encoding method (use Unicode UTF-8) for all languages.
  8. Click OK.
  9. File > Save.

This file now contains an embedded or internal style sheet. If these styles apply only to this page, keep this as is. If you want to apply these styles to other pages, see next lesson to convert to an external style sheet.

To Use the Insert Panel

The Insert Panel is divided into categories: HTML, Form, Templates, Bootsctrap Components, jQuery Module, jQuery UI, Favorites, and Hide Labels. The HTML category may be used to insert Div, Image, Paragraph, Heading H1...H6, Table, etc.

  1. Create Heading Text
    1. In the Insert panel, choose the HTML category.
    2. Click on Heading:H1
    3. Type "My Home Page" as the heading.
    4. Save the file.
    5. In the Properties Inspector, click on CSS and click on the Align Center icon.
    6. Click on Show Code View in the Document toolbar and switch back to Show Design View. Do this often.
    7. Click at the end of the line and hit the Enter key to go to the second line. Type your name.
    8. In the Insert panel, choose the HTML category and click on Heading:H3.
  2. Create a Link and Preview It in a Browser
    1. Highlight your name.
    2. In the Property inspector, click on HTML and type "http://sites.fitnyc.edu/users/gene_chin" in the Link box. This is an example of an absolute address. What happened to the H3 heading? It is no longer black and large. It is now an anchor tag (a) and has the attributes of a.
    3. Save the page.
    4. Click on the Preview/Debug in Browser button (in bottom right corner) and choose a browser or hit F12.
  3. Insert an Image
    1. Hit Enter key to go to the next line.
    2. In the Insert panel, click on HTML category.
    3. Click on Image.
    4. Choose an image from your assets.
    5. Switch to Live view and make sure the image is selected (the heads up display should have img. the DOM should have img highlighted, and the tags above the Properties Inspector should img highlighted.
    6. Click on CSS Designer panel. Drag the left edge of the CSS Designer panel until 2 columns appear.
    7. Click on + to add a new selector. It appears as the default "body p img". Change it to "img"
    8. Uncheck the Show Set button.
    9. Click on the Border property and set the width to 0.

To Create a Second Page

  1. Open index.html (if it is not already open) as a model to create the second page.
  2. Change the word "Home" to "Second" to read "My Second Page" now.
  3. Delete Your name and the image.
  4. Type the word "Home." after the heading.
  5. Select the Date object in the Form category of the Insert panel.
  6. Save this page as secondpage.html and click on the Refresh icon in the Files panel if you don't see the page here. Try out the calendar.

To Link to Another Page

Using text
  1. Highlight the word "Home" on the second page.
  2. Drag the Point to File icon to index.html.
  3. Save the file.
  4. Hit F12 to test it in a browser.
Using an Image
  1. Select the image on the Home Page by clicking on it.
  2. In the Property inspector on the right side of the Link box drag the Point to File icon to the file you wish to link to (in this case, the second page).
  3. In the Alt box, type alternate text for the image. The Alternate text will appear if the image doesn't and is also used by screen readers to aid a visually-impaired person.
  4. Save the file.
  5. Click on the Preview button or hit F12.
  6. Close everything for this exercise.
