To Create a Rollover

Perform the following steps and save the file as ex10a.html in the classwork/day10 folder.

  1. Launch Dreamweaver.
  2. Define your site. See Dreamweaver Lesson 1.
  3. File > New > New Document > HTML > None > Create.
  4. In the Insert panel, click on HTML category.
  5. Scroll all the way down to Rollover Image. Click on it.
  6. For Original Image, click on the Browse button to select your first image.
  7. For Rollover Image, click on the Browse button to select your second image, preferably with the same width and height as the Original image. If not, the second image will be distorted to conform with the first image's dimensions.
  8. For Alternate Text, type appropriate text to appear before the image is shown.
  9. For When Clicked, Go to URL, type a link if the user clicks on the rollover (optional).
  10. Save the file as ex10a.html in the classwork/day10 folder.
  11. Preview in a browser.
  12. Close this file.

To Create an Image Map (Hotspots)

Perform the following steps and save the file as ex10b.html in the classwork/day10 folder.

  1. File > New > Blank Page > HTML > none > Create.
  2. In the Insert panel, click on HTML category.
  3. Click on Image.
  4. Choose an image to be used for the image map (also known as hot spots).
  5. Save the file as ex10b.html in the classwork/day10 folder.
  6. Make sure the image is selected. Your image is selected if you see a thumbnail of the image and the word "Image" in the upper left corner of the Properties inspector.
  7. In the Map text box of the Properties inspector, type in a name for the map. This name will appear as the map attribute of the map tag and also as the usemap attribute of the img tag.
  8. Select a Hotspot tool (rectangle, oval, or polygon) and drag over a portion of the image to create a "hotspot".
  9. In the Link text box, do any one of the following:
    • Type the URL for the hotspot.
    • Drag the "Point to File" tool to the file in the Files panel that you want to use as the URL.
    • Click the "Browse for File" button to look for the file that you want to use as the URL.
  10. In the Target text box, select the window that you want to linked file to appear.
  11. In the Alt text box, enter the alternative name for the hotspot.
  12. Save the file.
  13. Preview in a browser.

To Convert an Embedded Style to an External Style

If you have more than a few pages, it is better to use external style sheets.

  1. Open a document that contains an embedded style you want to convert.
  2. In Code view highlight all the rules by dragging from the first one (such as body) to the last one, i.e., everything between <style> and </style>.
  3. Right-click any of the selected rules and choose CSS Styles, then click on Move CSS Rules.
  4. In the Move To External Style Sheet dialog box, select A New Style Sheet. Then click OK.
  5. In the Save Style Sheet File As dialog box, click Site Root, browse to the folder you are using, and save the file as a name your have chosen, such as styles.css. Click Save.
  6. Notice the rules for the embedded style have been removed and the link to the external style sheet has been added. Click on the CSS Designer panel.
  7. The CSS Designer panel shows <style> and styles.css. <style> refers to the previous embedded style sheet, which is now empty, whereas styles.css is the new external style sheet you have created. We no longer need <style> so select it and hit the Delete key.

To Link an External Style Sheet to a Page

Suppose we want both the rollover html (ex10a.html) and the hotspots html (ex10b.html) to have the same styles as index.html. We need to link the external style sheet to these two pages.

  1. Open ex10a.html.
  2. Format > CSS Stles > Attach Style Sheet.
  3. Select the style sheet you wish to have linked.
  4. Save the file.
  5. Press F12 to view in a browser.
  6. Repeat steps 1 to 5 for ex10b.html.
  7. Close all files

To Insert an Email Link

  1. To insert an email link highlight the word(s) that you want to set as a link to your email.
  2. Click on the Insert Panel.
  3. Choose the HTML category.
  4. Click on Email Link.
  5. When you are prompted, type in your email address.
