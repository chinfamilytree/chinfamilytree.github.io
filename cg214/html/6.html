iframe

The iframe tag creates an inline frame to contain another HTML document. For example:

<iframe src="f1.html"></iframe>

You can target an iframe by giving it a name:

<iframe src="f1.html" name="if"></iframe>

A width and height may be specified:

<iframe src="f1.html" name="if" width="570" height="440"></iframe>

The global id attribute is supported. The id gives this element a unique id. The global style and title attributes are also supported. Style is an inline style below and title gives more information about this element.

<iframe src="f1.html" name="if" width="420" height="620" 
        id="if" style="border-style:inset;" title="The name of this iframe is 'if'"></iframe>

The value in the border-style property may be none, dotted, dashed, solid, double, groove, ridge, inset, outset, or hidden.

iframe Sample

Here's a complete example with a table one to contain the thumbnails across the top of the page and the other to contain the iframe at the bottom of the page.

<!doctype html>
<html>
<head>
<title>Paintings</title>
<meta charset="utf-8">
<style type="text/css">
body
{ background-color:#c4c3d0;
margin-left:10px;
margin-top:10px;
margin-right:0px;
margin-bottom:0px;
}
img {border:0;}
</style>
</head>
<body>
<p>
<a href="boreas.html" target="if1"><img src="boreas-t.jpg" title="boreas-t"></a>
<a href="circe.html" target="if1"><img src="circe-t.jpg" title="circe-t"></a>
<a href="lady.html" target="if1"><img src="lady-t.jpg" title="lady-t"></a>
<a href="ophelia.html" target="if1"><img src="ophelia-t.jpg" title="ophelia-t"></a></p>
<iframe id="if1" name="if1" src="boreas.html" title="Paintings" width="353" height="520" style="border:inset;">
</iframe>
</body>
</html>

Sounds

There are several sound formats such as AU, AIFF, WAV, MIDI, and MP3. The sampling rate of AU is low (8-bit) and thus produces poor quality sounds. Apple developed the AIFF format for the Macintosh. IBM and Microsoft developed the WAV format for the PC. MIDI (Musical Instrument Digital Interface) is a standard protocol for connecting electronic musical instruments to computers. MP3 is a compressed format that eliminates sounds which the human ear can't hear. All these formats are proprietary formats. Some browsers currently support Ogg Vorbis (Ogg for short) sound format, which is a free open source software supported by HTML5. As of February 9, 2015, Ogg is supported by Chrome, Firefox, and Opera. Support can be added to Safari by installing an add-on to desktops, but not iOS. Ogg is not supported by Internet Explorer.

See a comparison of Ogg Vorbis and MP3.

Audio Element

Below is an example of the HTML5 audio element. The ogg file, thehappyfarmer.ogg, is 910,845 bytes and plays for 49 seconds. You can download other ogg files from http://www.piano-midi.de/ogg.htm. The equivalent midi file is 3,187 bytes - 40 seconds. Hear it.

<!doctype html>
<html>
<body>
<audio controls>
  <source src="happyfarmer.ogg" type="audio/ogg" />
</audio>
</body>
</html>

Variations

1. Audio element with controls (normal)
<audio controls>
  <source src="happyfarmer.ogg" type="audio/ogg" />
</audio>
2. Audio element with controls, but muted
<audio controls muted>
<source src="happyfarmer.ogg" type="audio/ogg">
</audio> 
     If you want to hear the music...
       Drag the volume control to the right.
 		Then click on the Play button.
3. Audio element with controls and autoplay on

<audio controls autoplay>
  <source src="happyfarmer.ogg" type="audio/ogg" />
</audio>

4. Audio element with no controls and autoplay on

<audio autoplay>
  <source src="happyfarmer.ogg" type="audio/ogg" />
</audio>

5. Audio element with no controls, autoplay on and loop on
(acts like background sound, playing continuously)

<audio autoplay loop>
  <source src="happyfarmer.ogg" type="audio/ogg" />
</audio>

WAV

Safari does not recognize ogg but it does recognize WAV, so if you add a WAV file, Safari will play the first one it recognizes. Internet Explorer doesn't support the <audio> element, so it will therefore display some error message. Browsers will skip the ones it can not find and attempt to play the next one on the list. In the example below, happy_farm.ogg is not found and the next file, gust.wav, will be played.

<!doctype html>
<html>
<body>
<audio controls>  
  <source src="happy_farm.ogg" type="audio/ogg" />
  <source src="gust.wav" type="audio/wav" />
</audio>
</body>
</html>

embed Element

You could <embed> the any sound file like this:

<embed src="harp.wav">

The <embed> method works in Chrome, Firefox, Internet Explorer, and Safari but not in Opera. Play it..

Hyperlink

You can hyperlink to any sound.

<!doctype html>
<html>
<head>
<meta charset="utf-8"> 
<title>Using Hyperlink to a Sound</title>
</head>
<body>
<h2>Using A Hyperlink</h2>
<p><b><a href="happyfarmer.ogg">The Happy Farmer (ogg)</a></b></p>
<p><b><a href="harp.wav">Harp (wav)</a></b></p>
</body>
</html>
