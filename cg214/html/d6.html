To Create a Table

Perform the following steps and save the file as ex11.html in the classwork/day11 folder.

  1. Launch Dreamweaver.
  2. Define your site. See Dreamweaver Lesson 1.
  3. File > New > New Document > HTML > None > Create.
  4. Click on the Insert panel > HTML category.
  5. Click on Table.
  6. In the dialog box, enter information for Rows, Columns, Table width, Border thickness, Cell padding, and Cell spacing.
  7. Choose the type of Header you want: None, Left, Top, or Both (left and top).
  8. Enter a Caption.
  9. Choose an alignment for the Caption in relation to the table: default (top), top, bottom, left, or right.
  10. Summary is a description of the table and does not appear in the user's browser. It is used by screen readers.
  11. Click OK.
  12. To fill a table with content you can type directly into the table or paste text into cells. Use the Tab key to move forward to the next cell. Doing this in the table's last cell will automatically add another row. Use Shift+Tab to move to a previous cell. You can also use Arrow keys to move within cells and between cells.
  13. Select the cells you want to merge (span). In the Property inspector, click on the "Merge selected cells using spans" button in the lower left corner.
  14. Select the cells you wish to be Headers by dragging across them. In the Property inspector, click on the check box for Header.
  15. Select the cell(s) you want to change the alignment. In the Property inspector, click on any of 4 alignment icons: Left, Center, Right, or Justify.
  16. Add an image to a table by clicking in the cell where you want the image and then choose Insert > Image. Select the image you want and click OK.
  17. You can create a whole table inside a table's cell. This is called nesting tables. Click the cell where you want the new table to appear. Then choose Insert > Table.
  18. You may also use a preset design for a table. Select the table and then click choose Commands > Format Table. Choose a design scheme from the list on the left.
  19. You can sort a table by column. You cannot sort tables that contain merged cells (colspan or rowspan attributes).
    1. Select the table and choose Commands > Sort Table.
    2. Select the columns in the Sort By option. Select alphabetically or numerically in the Order option.
    3. Do a secondary sort by specifying the sorts under Then By.
    4. Be default, the first row is not included in a sort because it is usually a header row. To include the first row in a sort check Sort Includes First Row.
  20. Besides changing the size of a table or cells in the Property inspector you can also click and drag the handles on a table. Leave the W field blank to let the browser and Dreamweaver determine the width based on the contents of the cells. You can also clear the width and height by choosing Modify > Table and then Clear Cell Widths or Clear Cell Heights. There are also icons in the lower left corner of the Property inspector to perform the same functions.
  21. Use Modify > Table to add or delete rows and columns. To add or delete from the bottom and right of a table, first select the entire table by clicking the table's upper left corner or choosing Modify > Table > Select Table. Make adjustments in the Property inspector.
  22. You can copy and paste multiple cells at a time preserving the formatting (Edit > Copy or Cut and then Edit > Paste). Remove just the cell content by selecting one or more cells and pressing the Delete key or choose Edit > Clear. You can also paste copied cells outside the table to create a new table.
  23. Save the file as ex11.html in the classroom/day11 folder.
  24. Preview in a browser.

Using IFrames

The iframe tag creates an inline frame to contain another HTML document. To create an iframe:

  1. Launch Dreamweaver.
  2. Define your site. See Dreamweaver Lesson 1.
  3. File > New > New Document > HTML > None > Create
  4. Insert > HTML > IFrame.
  5. Click on Code or Split View. You will see <iframe></iframe>. This is all Dreamweaver gives you.
  6. In the beginning iframe tag click between "e" and ">". Hit the spacebar. Dreamweaver prompts you to select attributes. Double click on the attribute you want (such as "id") and enter the value for that attribute.
  7. Hit the spacebar again and Dreamweaver prompts you for another attribute. Double click on the attribute you want and enter the value for that attribute, etc. Some attributes to consider besides id are name, width, height, and src.
