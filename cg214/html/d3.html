CSS Transitions

CSS transitions allow you to change property values smoothly (from one value to another), over a given duration.

  1. Click on an image to have CSS Transitions done on it.
  2. Window > CSS Transitions.
  3. Click on + to add a new transition.
  4. In Target Rule, choose a target from the list or make a new one.
  5. Let's make a new one, call it ".smooth".
  6. In Transition On: select Hover.
  7. In Duration, set it to 1s (1 second). [ms is milliseconds.]
  8. Don't set anything for Delay.
  9. In Timing Function, select ease-in-out. [See this article for differences in all the timing functions.]
  10. In the Property box, click on + in lower right ocrner and select "transform".
  11. In End Value, type rotate(30deg) scale(1.8). [rotate(30deg) means rotate clockwise 30 degrees; a negative value indicates counterclockwise; scale(1.8) means enlarge it by 80%; a value less than 1.0 means to decrease the size of the image.]
  12. Click on "Create Transition"
  13. If you apply a class of "smooth" to another image on this page, it will perform the same transition.
  14. Click on the "x" in the upper right corner to the New Transitions dailog box to close it.
  15. For more information, see https://helpx.adobe.com/dreamweaver/how-to/use-the-css-transitions-panel.html.

To Attach a Behavior

Save all your work in the classwork/day11 folder.

You can attach behaviors to the entire document, or to links, images, form elements, layers, or almost any HTML element. You can specify more than one event to trigger a behavior, and more than one action for each event. Actions occur in the order listed in the Action panel of the Behavior inspector.

  1. Select an object. To attach a behavior to the entire page, click the <body> tag in the tag selector at the bottom left of the Document window.
  2. Choose Window > Behaviors to open the Behavior panel. The tag of the selected object appears in the title bar at the top of the panel.
  3. Click the Events pop-up menu (+ button) and choose an event to trigger the behavior. Different events appear depending on the selected object. If the expected events don't appear, make sure that the correct object is selected. If you're attaching a behavior to an image, several events such as onMouseOver and onMouseDown appear in parentheses. These events are only available for links. When you choose one of them, Dreamweaver adds an <a> tag to the image to define a link.
  4. Choose an action from the Actions pop-up menu. Actions that don't work in the current document are not available on the menu. For example, the Play Timeline action is not available if the document has no timeline. Depending on the chosen action, a dialog box appears to display parameters and instructions for the action.
  5. Enter parameters for the action and click OK. All included actions work in 4.0 or later browsers. Some actions do not work on older browsers.

Sample Behavior 1: Swapping an Image

  1. Make thumbnails of large images.
  2. In Dreamweaver, insert all thumbnails, giving each a unique ID in the Properties Inspector.
  3. Click on first thumbnail.
  4. Click on Window/Behaviors.
  5. Click on + to add a behavior.
  6. Click on Swap Images.
  7. Browse for the image to swap. This is the corresponding larger image to the selected thumbnail.
  8. Click on the larger image.
  9. Click on OK.
  10. Click on OK again.
  11. Go to Code view.
  12. IMPORTANT: Remove all references to Width and Height.
  13. Save the html file.

Sample Behavior 2: Combination of Rollover and Swap Image

  1. Prepare your images: large, thumbnails, and rollovers. This exercise has 4 sets (12 images total).
  2. Insert thumbnails (images with "-th" at the end of the filename) into position, such as across the top. Give each thumbnail an "Alternate text" and a unique ID in the Property inspector. Set the border to 0.
  3. Insert large image in the next line. Give "Alternate text" and unique ID.
  4. Important: Go to Code view and remove references to width and height for this large image.
  5. Go back to Design view.
  6. Click on the first image.
  7. Window > Behaviors. (Shift+F4)
  8. Click on first thumbnail.
  9. Click on the "+" sign in the Behavior panel and click on Swap Image.
  10. In the Swap Image dialog box, browse for the image you want to swap for the selected one, such as a rollover image. Do not click OK yet.
  11. In the same Swap Image dialog box, click on the ID for the large image, browse for the large image you want to swap for the selected one. Now click OK.
  12. Click on the next image and repeat the above three steps until you have done all the images.
  13. Click on Live View to test it. (Note: to get out of Live View click on it again.)
  14. Save your html file.

See a demo of this behavior.
